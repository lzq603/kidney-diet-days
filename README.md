# 🫘 肾友食光

本项目是一款专为慢性肾脏病患者设计的智能饮食管理微信小程序，基于大语言模型和计算机视觉技术，为用户提供个性化食谱安全性分析、食物成分识别、每日营养摄入统计等功能，辅助患者进行科学合理的饮食控制，提升生活质量。

## ✨ 功能特点

### 🍽️ 核心功能

- **AI 食谱分析** - 输入菜品名称，AI分析食谱安全性
- **AI 营养师对话** - 与AI营养师进行自然语言对话咨询
- **拍照识别** - 支持拍照识别食材进行分析
- **1000+食材库** - 常见食材安全等级和营养数据
- **营养摄入统计** - 钾、磷、钠、蛋白质等核心指标

### 📊 记录管理

- **每日饮食记录** - 记录每一餐的食物和摄入量
- **历史记录** - 查看历史饮食数据
- **营养报告** - 每日营养摄入统计
- **检查报告** - 支持上传检查报告并AI分析

### 🏥 个性化设置

- **肾功能分期** - 用户设置自己的肾功能分期
- **肾病类型** - 支持多种肾病类型
- **基础信息** - 身高、体重、年龄等
- **营养限制** - 根据分期自动调整每日营养限制

## 🛠️ 技术栈

### 前端

- 微信小程序原生开发
- 自有后端
- WebSocket 实现 AI 流式响应

### 后端

- Django + Django REST Framework
- DeepSeek AI API
- 数据存储（用户数据、食谱库等）

### AI 能力

- DeepSeek 大语言模型
- Qwen 视觉模型
- 支持多模态分析（文字+图片）

## 📖 使用说明

### 1. 首次使用

1. 打开小程序后，点击"我的"
2. 设置个人信息和肾功能分期
3. 完善身高、体重等基础数据
4. 开始使用饮食分析功能

### 2. 记录饮食

1. 在首页对话或"食谱"页搜索
2. 查看AI分析结果和建议
3. 调整摄入克数，记录到当日饮食
4. 在"记录"页查看统计数据

### 3. 拍照识别

1. 点击"食谱"页的"拍照分析"
2. 拍摄或选择食物照片
3. AI 自动识别并分析
